RECRUTEMENTS

Nous recrutons des modérateurs. Si vous souhaitez postuler, je vous invite à remplir ce formulaire ici Formulaire recrutement
( https://docs.google.com/forms/d/e/1FAIpQLScG4wyevXHKiYoeLgAWEwKPShqHJKdhMOezdoP8mosk02fzDw/viewform?usp=sf_link )


Pour tout les postes de "Directeur" (Directeur Fortnite, staff, rocket league, valorant etc...), les recrutemements se font sur notre Discord
, une fois dans celui-ci, rendez vous dans le channel "rc staff", ouvrez un ticket et déposez votre plus belle candidature !


 Pour les postes de "Responsable", "Graphiste, "Content Creator, "streamer", ou encore "monteur"
"c'est comme les Directeurs, il faut se rendre sur notre
Discord, une fois dans celui-ci, rendez vous dans le channel "rc staff", ouvrez un ticket et déposez votre plus belle candidature !


Ensuite, voici comment postuler et les critères pour devenir joueur FTN :

- Avoir 500 de pr (Joueur officiel) ou 200 de pr (Joueur académique)
- Avoir 14 ans minimum
- Avoir un bon PC et une bonne co

Pour postuler, vous devrez vous rendre sur notre Discord, une fois dessus, rendez vous dans le channel "rc joueur", ouvrez un ticket et posulez dedans !


Pour ce qui est des recrutements pour devenir joueur Rocket League, voici les critères et comment se faire recruter :

- Être minimum CHAMPION II
- Avoir 14 ans minimum

Pour postuler, vous devrez vous rendre sur notre Discord, une fois dessus, rendez vous dans le channel "rc joueur", ouvrez un ticket et posulez dedans !
